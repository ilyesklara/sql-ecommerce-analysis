# sql-ecommerce-analysis
This project focuses on cleaning and analyzing an e-commerce sales dataset using SQL. The goal was to simulate a real-world data analyst workflow including data cleaning, validation, and exploratory analysis.

Steps performed

• Data inspection
• Fixing column names
• Cleaning inconsistent values
• Handling invalid numeric values
• Recalculating totals
• Removing duplicates
• Exploratory data analysis using SQL
